#  Bright Coffee Shop Sales Analysis

---
## 📊 Overview

This project is an end-to-end data analytics case study focused on analyzing coffee shop sales data to uncover meaningful business insights.

The goal of this analysis was to understand sales performance, identify high-performing products, analyze customer purchasing patterns, and determine peak sales periods. By using structured data analysis techniques, the project demonstrates how raw data can be transformed into actionable insights for better decision-making.

---

##  Key Objectives

* Analyze total revenue and overall sales performance
* Identify best-selling products and categories
* Determine peak hours for sales activity
* Understand customer purchasing trends
* Evaluate store and product performance

---

## Process

The project followed a structured data analysis workflow:

1. Data Exploration – Understanding the dataset and its structure
2. Data Cleaning – Preparing data for analysis
3. Data Analysis (SQL) – Writing queries to extract insights
4. Data Visualization (Excel) – Creating pivot tables and charts
5. Presentation – Communicating findings using PowerPoint

---

##  Key Insights

* Coffee products generate the highest revenue
* Sales peak during specific hours of the day
* Certain product categories outperform others
* Some products show lower demand and require improvement
